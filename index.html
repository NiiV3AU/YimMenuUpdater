YMU: YimMenu. Simple. Up-To-Date.

The smart launcher for YimMenu. Automatic updates, easy injection and Lua management – all in one place.

Download YMU v1.1.5 (.exe)

Key Features

Everything you need for YimMenu, in one tool.

Always Up-to-Date

YMU automatically checks for YimMenu updates and notifies you. One click is enough to update.

Easy Injection

No separate injection tool needed anymore. Start GTA V and inject YimMenu directly from YMU.

Lua Script Management

Manage your Lua scripts, find new ones to try out, and easily enable / disable them in the settings.

Modern UI

An appealing and intuitive user interface (thanks to PySide6) with Light & Dark Themes.

Integrated Help

Feeling lost? buttons in every tab help you use the tool optimally.

Self-Updater

YMU itself also stays up-to-date if desired, so you always have the latest features.

Look & Feel

Modern, clean, and customizable.

How it Works

Ready in just a few steps.

Download & Launch

Download ymu.exe and launch it.

YimMenu Download

In the 'Download' tab, you can download or update YimMenu to the latest version.

Start GTA V & Inject

Switch to the 'Inject' tab, start GTA V via the button (optional), and inject YimMenu.

You can find more detailed information via the buttons directly in the YMU application.

Frequently Asked Questions (FAQ)

Is YMU safe?

YMU itself does not contain malicious software.

However, using mod menus like YimMenu is always at your own risk.

Please note the warnings.

How do I install the dependencies if I use the source code?

You can install the dependencies using pip: 

pip install PySide6 pywin32 psutil pyinjector requests packaging
